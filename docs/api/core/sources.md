## `fromArray`

#### Signature
<pre>function fromArray&lt;T&gt;(array: ArrayLike&lt;T&gt;): <a href="_index.md#source-interface">Source</a>&lt;T&gt;;</pre>

Creates a Source from the given array/array-like. The values of the array will be synchronously emitted by the created source upon each subscription.

#### Parameters

| <p>Parameter</p> | <p>Type</p> | <p>Description</p> |
| --- | --- | --- |
| <p>`array`</p> | <p><code>ArrayLike&lt;T&gt;</code></p> | <p>The array/array-like to iterate over.</p> |

#### Returns

| <p>Type</p> | <p>Description</p> |
| --- | --- |
| <p><code>[Source](_index.md#source-interface)<!---->&lt;T&gt;</code></p> | <p>The created source.</p> |

#### Example

```ts
pipe(fromArray([1, 2, 3, 4]), subscribe(Sink(console.log)));
// Logs:
// Push(1), Push(2), Push(3), Push(4)
// End
```

#### Example

```ts
pipe(
    fromArray({ length: 5, 0: 'foo', 3: 'bar' }),
    subscribe(Sink(console.log))
)
// Logs:
// Push('foo'), Push(undefined), Push(undefined), Push('bar'),
// Push(undefined)
// End
```
